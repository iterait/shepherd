    shepherd.errors.api¶

    Exceptions¶

    • ApiServerError: An exception raised when the server encounters an error

    • ApiClientError: An exception raised when the user of the API makes an invalid request

    • UnknownSheepError: Exception raised when application attempts to use a sheep with an unknown id.

    • UnknownJobError: Exception raised when a client asks about a job that is not assigned to this shepherd.

    • StorageError: Exception raised when application encounters some issue with the minio storage.

    • StorageInaccessibleError: Exception raised when the remote storage is not accessible at the moment

    • NameConflictError: Exception raised when a client chooses a job ID that was already used
