    shepherd.runner.run(model, dataset, stream_name, payload)[source]¶

    Get the specified data stream from the given dataset, apply the given model on its batches and return the results.

    The components have to be emloop compatible with:

    • dataset having method named [stream_name]_stream taking the payload and returning the stream

    • (optional) dataset having method named postprocess_batch taking both the input and output batches and

      returning the post-processed batch

    Parameters

    • model (AbstractModel) – emloop model to be run

    • dataset (AbstractDataset) – emloop dataset to get the stream from

    • stream_name (str) – stream name

    • payload (Any) – payload passed to the method creating the stream

    Return type

    Mapping[str, Sequence[Any]]

    Returns

    result batch (if the stream produces multiple batches its the concatenation of all the results)

    shepherd.runner.n_available_gpus()[source]¶

    Return the number of NVIDIA GPU devices available to this process.

    Note

    This method attempts to parse (in-order) CUDA_VISIBLE_DEVICES env variable (bare sheep) and NVIDIA_VISIBLE_DEVICES env variable (docker sheep). If none of these variables is available, it lists GPU devices in /dev.

    Return type

    int

    Returns

    the number of available GPU devices
