    static recv(socket, expected_message_types=None, noblock=False)[source]¶

    Receive, decode and return a message from the given socket.

    Parameters

    • socket (Socket) – socket to receive the message from

    • expected_message_types (Optional[Sequence[type]]) – a sequence of expected message types (optional)

    • noblock (bool) – do not block on recv call and throw if there is no message

    Raises

    • MessengerError – if receiving fails

    • UnknownMessageTypeError – if the received message is of unknown type

    • UnexpectedMessageTypeError – if the received message type is not expected

    Return type

    Union[InputMessage, DoneMessage, ErrorMessage]
